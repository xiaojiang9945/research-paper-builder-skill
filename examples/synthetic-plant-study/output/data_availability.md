# Data Availability Statement

All data in this demo are synthetic and are included in the repository:

- `examples/synthetic-plant-study/input/synthetic_germination_data.csv`
- `examples/synthetic-plant-study/output/source_data_figure1.csv`
- `examples/synthetic-plant-study/output/supplementary_table_s1_summary.csv`

No real biological samples, human participants, patient data, unpublished project data, institutional identifiers, or confidential collaborator materials are included.

For a real manuscript, replace this statement with repository accession numbers, archive DOI, supplementary table references, code release information and any access restrictions required by the journal.
