# Literature Intake Status

This demo does not claim that 200 real papers were searched or read. It uses a synthetic citation set only to demonstrate manuscript mechanics: citation placement, reference-list matching, and citation-audit output.

## Real Manuscript Requirement

For a real high-ambition manuscript using this skill, the writing gate should require:

- at least 200 records screened from relevant databases and citation trails;
- full-text reading notes for the core literature set;
- extraction of figure architecture, methods precedent, claim boundaries and validation expectations;
- a saved literature matrix with relevance tier, citation role and inclusion decision;
- verified reference metadata before final manuscript drafting.

If the literature intake is incomplete, the skill should produce a literature plan, reading matrix and outline instead of a polished final manuscript.

## Demo Substitution

The demo substitutes a small synthetic citation set:

- `references_demo.md`: synthetic reference list for citation-matching demonstration;
- `citation_reference_audit.csv`: one-to-one mapping between in-text citation keys and reference-list entries;
- `manuscript_draft.md`: long-form manuscript draft that shows where citations would appear in a real paper.

These files are not literature evidence.
